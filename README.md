# trafficracer
Programming Technologies and Environment project

### Trafficracer game, evade the holes!

Controls:
  - WASD for movement
  - ESCAPE to give up
  
## Create a new user and achieve a new highscore!


After cloning:
To build the jar:
mvn clean package

To run the jar:
java -jar target/trafficracer-1.0-SNAPSHOT-shaded.jar
